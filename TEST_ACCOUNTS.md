## 自动登录

访问WarkAI网站时，系统会**自动登录第一个测试账号** (`test_pilot_1`)。您可以：

1. **直接使用** - 无需任何操作，已自动登录
2. **切换账号** - 在浏览器控制台执行：
   ```javascript
   switchTestAccount('test_pilot_2');
   ```
3. **查看当前账号** - 在浏览器控制台执行：
   ```javascript
   getCurrentAccount();
   ```

## 功能说明

### 驾驶员模块
点击右上角的 "👤 驾驶员模块" 可以访问：
- **个人中心** - 查看和管理个人资料
- **意见反馈** - 提交反馈和建议

- **返回首页** - 返回主页面

### 游戏页面
访问 `/game.html` (中文) / `/game-en.html` (英文) / `/game-ja.html` (日文) 可以玩高达模拟游戏。

点击右上角的语言切换器可以在中文、英文、日文之间切换。

## 技术实现

测试账号系统通过 `js/test-account.js` 实现，包含以下功能：

- **自动登录**: 页面加载时自动登录测试账号
- **账号管理**: 获取、切换、注销账号
- **本地存储**: 使用 localStorage 保存登录状态
- **多语言支持**: 支持中文、英文、日文账号名称

## 注意事项

1. **测试账号数据** - 所有测试账号数据存储在浏览器本地，刷新页面后会重置
2. **隐私保护** - 测试账号不涉及真实用户数据
3. **安全性** - 测试账号仅用于演示目的，不应在生产环境中使用
